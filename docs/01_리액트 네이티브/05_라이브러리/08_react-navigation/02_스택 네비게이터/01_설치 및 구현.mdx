
스택 네비게이터 사용을 위해서는 아래의 패키지 설치가 필요해요.

## 설치
```
yarn add @react-navigation/native
yarn add react-native-screens react-native-safe-area-context react-native-gesture-handler
yarn add -D @types/react-native-gesture-handler
yarn add @react-navigation/native-stack
```

## 구현

```ts
import { createNativeStackNavigator } from '@react-navigation/native-stack';

const Stack = createNativeStackNavigator();

const RootNavigation = () => {
  return (
    <Stack.Navigator>
      <Stack.Screen name={routes.main} component={MainScreen} />
    </Stack.Navigator>
  );
};

export default RootNavigation;
```
