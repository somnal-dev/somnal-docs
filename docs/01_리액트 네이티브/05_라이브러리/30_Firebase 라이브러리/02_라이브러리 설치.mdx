
아래의 명령어를 실행하여 `React-Native용 Firebase 라이브러리` 설치를 진행할 수 있어요.

```bash
yarn add @react-native-firebase/app
```

iOS의 경우 `pod install` 명령 실행이 필요해요.<br/>
만약, 그래도 오류가 발생한다면 `pod install --repo-update` 명령어를 실행해주세요.

yarn add @react-native-firebase/auth

yarn add @react-native-firebase/storage
