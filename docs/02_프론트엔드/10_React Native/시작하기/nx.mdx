## Nx로 시작하기

### 1. 빈 프로젝트 생성

```bash
npx create-nx-workspace@latest <프로젝트명> --preset=apps --packageManager=yarn
```

### React Native 앱 생성
```bash
cd <프로젝트명>
yarn add -D @nx/react-native
```
```bash
nx generate @nx/react-native:app <앱 이름>
```

### 라이브러리 생성
```bash
nx generate @nx/react-native:lib components
nx generate @nx/react-native:lib core
```

## 실행
```bash
# iOS 실행
nx run mobile:run-ios

# Android 실행
nx run mobile:run-android

# Metro 서버만 시작
nx run mobile:start
```
