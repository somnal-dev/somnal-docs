## 활용 1. DOM 객체에 대한 직접적인 참조

버튼 클릭 시 input에 포커스를 주는 예시

<UseRefTest />

```typescript jsx
import { useRef } from 'react';

function MyComponent() {
    // input 객체에 대한 useRef 선언
    const inputRef = useRef();

    const handleClick = () => {
        inputRef.current.focus(); // input DOM 요소에 접근
    };

    return (
        <>
            {/*input 객체 참조 바인딩*/}
            <input ref={inputRef} />
            <button onClick={handleClick}>Focus Input</button>
        </>
    );
}
```

## 활용 2. 렌더링과 관계없는 값의 저장 (재렌더링 하고싶지 않을 경우)

값이 변경돼도 컴포넌트는 리렌더링되지 않음.

<UseRefTest2 />

```typescript jsx
import {useRef} from "react";
const countRef = useRef(0);

function handleClick() {
  countRef.current += 1;
  console.log(countRef.current); // 값은 변하지만 컴포넌트는 리렌더링되지 않음
}
```
