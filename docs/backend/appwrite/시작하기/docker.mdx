# docker 로 시작하기

```bash
docker run -it --rm \
    -p <포트번호>:80 \
    --volume /var/run/docker.sock:/var/run/docker.sock \
    --volume "$(pwd)"/appwrite:/usr/src/code/appwrite:rw \
    --entrypoint="install" \
    appwrite/appwrite:1.7.4
```
AppWrite 내부 포트를 30000 바꾸고,
외부 포트번호를 30000 으로 하고싶다면
```bash
docker run -it --rm \
    -p 30000:30000 \
    --volume /var/run/docker.sock:/var/run/docker.sock \
    --volume "$(pwd)"/appwrite:/usr/src/code/appwrite:rw \
    --entrypoint="install" \
    appwrite/appwrite:1.7.4
```

## 실행 및 중지

위의 명령어를 실행하면 현재 디렉토리에 appwrite 폴더가 생긴다.<br/>
appwrite폴더 밑의 docker-compose.yml 파일로 appwrite를 시작 및 중지할 수 있다.

### 실행
```bash
docker-compose up -d
```

### 중지
```bash
docker-compose down
```
