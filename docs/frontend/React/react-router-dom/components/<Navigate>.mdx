useNavigate의 컴포넌트화 버전이다.

## 예시 및 옵션
```js
<Navigate
  to="이동할 경로"          // 이동할 대상 경로. 문자열 또는 위치 객체 (예: '/home' 또는 { pathname: '/home', state: {...} })
  replace                   // true일 경우, history에 현재 기록을 남기지 않고 해당 경로로 대체함 (뒤로가기 시 현재 페이지는 건너뜀)
  state={{ ... }}           // 이동할 때 함께 전달할 상태값 (location.state로 접근 가능)
  relative="route"|"path"   // 상대 경로의 기준을 설정 ('route'는 현재 라우트 기준, 'path'는 URL 경로 기준)
/>
```
