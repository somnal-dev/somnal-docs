Node의 기본 패키지 매니저. package.json 기반으로 의존성을 설치함

<br/>

### 장점

- 초창기부터 사용해온 패키지 매니저이므로, 호환성이 굉장히 좋다.
- 생태계가 풍부한 편이다.

### 단점

- npm으로 패키지를 설치할 경우 서로의 패키지들이 의존성을 갖고 있기 때문에, 하나의 패키지에 문제가 발생한다면 다른 패키지에도 문제가 발생할 수 있다.
- node_modules 폴더가 차지하는 크기가 크다.


### 실행 예시

```bash
npm init
```
