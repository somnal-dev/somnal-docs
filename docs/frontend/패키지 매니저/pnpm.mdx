2017년 `Zoltan Kochan` 이라는 개발자가 출시한 패키지 매니저<br/>
performant npm 을 줄여서 pnpm 이라고 이름지었다.<br/>

## 장점
- 기존 npm의 프로젝트별로 node_modules를 설치했던 방식과 달리 global 저장소에 패키지를 한번만 설치하므로, 저장공간 절약 가능

## 단점
- 특정 패키지를 한번만 설치하기 때문에, 프로젝트별로 연결해버리면 호환성 문제가 생길 수 있음. `별도 버전관리 필요`


## 설치 방법
```bash
npm install -g pnpm@latest-10
```

혹은 homebrew를 통해 설치가능

```bash
brew install pnpm
```
